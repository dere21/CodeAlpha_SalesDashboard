# 📊 Sales Performance Dashboard using Python

## 📌 Project Overview

This project presents a Sales Performance Dashboard created using Python for data analysis and visualization. The dashboard analyzes retail sales data to identify important business insights related to revenue trends, product performance, regional sales distribution, category-wise sales, and profit trends.

The project was developed using Google Colab and Python visualization libraries including Pandas, Matplotlib, and Seaborn.

---

# 🎯 Objectives

- Analyze monthly sales revenue trends
- Identify top-performing products
- Compare regional sales performance
- Visualize sales distribution by category
- Analyze monthly profit trends
- Generate business insights through data visualization

---

# 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

# 📂 Dataset

The project uses the Superstore Sales dataset containing retail transaction information including:

- Order Date
- Sales
- Profit
- Product Name
- Category
- Region
- Customer Information

---

# 🧹 Data Cleaning and Preprocessing

The following preprocessing steps were performed:

- Converted date columns into datetime format
- Created Month-Year features for time-series analysis
- Prepared the dataset for visualization and analysis
